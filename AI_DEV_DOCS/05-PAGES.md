### URL Parameters

| Param | Type | Example | Description |
|-------|------|---------|-------------|
| `q` | string | `?q=jazz` | Search query |
| `category` | string | `?category=music` | Category slug |
| `date` | string | `?date=2025-02-14` | Specific date |
| `from` | string | `?from=2025-02-14` | Date range start |
| `to` | string | `?to=2025-02-28` | Date range end |
| `free` | boolean | `?free=true` | Free events only |
| `sort` | string | `?sort=date-asc` | Sort order |
| `page` | number | `?page=2` | Pagination |

### Data Requirements

```typescript
interface EventsPageProps {
  searchParams: {
    q?: string;
    category?: string;
    date?: string;
    from?: string;
    to?: string;
    free?: string;
    sort?: string;
    page?: string;
  };
}

// In page.tsx
const events = await getEvents({
  search: searchParams.q,
  categorySlug: searchParams.category,
  dateRange: parseDateRange(searchParams),
  isFree: searchParams.free === 'true',
  orderBy: parseSort(searchParams.sort),
  page: parseInt(searchParams.page || '1'),
  limit: 24,
});

const categories = await getCategories();
const totalCount = await getEventsCount(filters);
```

### SEO

```typescript
export async function generateMetadata({ searchParams }): Promise<Metadata> {
  const category = searchParams.category;
  const date = searchParams.date;
  
  let title = 'Events';
  if (category) title = `${capitalize(category)} Events`;
  if (date) title += ` on ${formatDate(date)}`;
  
  return {
    title,
    description: `Browse ${category || ''} events...`,
  };
}
```

### Slug Parsing

The slug format is `{event-slug}-{YYYY-MM-DD}`:
- `jazz-at-the-lake-2025-02-14`

```typescript
function parseEventSlug(fullSlug: string) {
  // Extract date from end (always YYYY-MM-DD format)
  const dateMatch = fullSlug.match(/-(\d{4}-\d{2}-\d{2})$/);
  if (!dateMatch) throw new Error('Invalid event slug');
  
  const date = dateMatch[1];
  const slug = fullSlug.replace(`-${date}`, '');
  
  return { slug, date };
}
```
